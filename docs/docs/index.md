# Welcome to Jyoti AI

Jyoti is your advanced AI Personal Assistant, designed to be polite, kind, and incredibly helpful.

## Features

*   **WhatsApp Integration**: Interact via your favorite messaging app.
*   **Persona Management**: "Donna" for the owner, "Professional PA" for guests.
*   **Memory**: Vector database stores all context securely.
*   **Proactive**: Initiates chats, summarizes emails, and manages tasks.
*   **Privacy**: Strict data isolation for the owner.
